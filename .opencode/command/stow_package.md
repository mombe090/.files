---
description: Create new folder a tools configuration with stow package manager.
---

Create new directory structure in the root of the project.

Follow the structure below:

```shell
$ARGUMENTS
   |- README.md
   |- .config
      |-$ARGUMENTS
```

Check in internet or github or if you are familliar with ths $ARGUMENTS tool you can create generate the README.md file with the basic information about the tool, installation instructions and usage.

> NOTE: it's alway important respect the markdown linting rules inside .markdownlint.jsonrc file in the root of the project.
